# 🤖 AI-Powered Resume Analyzer

An intelligent web application that analyzes resumes using AI and evaluates their compatibility with job descriptions. The system generates ATS-style scores and provides feedback to help improve resume quality.

---

## 🚀 Features

- 📄 Resume upload and analysis  
- 🤖 AI-based resume evaluation  
- 📊 ATS compatibility scoring  
- 🎯 Job description matching  
- 💡 Suggestions to improve resume quality  

---

## 🛠 Tech Stack

- ⚛️ React  
- 🟦 TypeScript  
- 🎨 Tailwind CSS  
- ☁️ Puter.js  
- 🤖 AI APIs  

---

## 📊 How It Works

Resume Upload
↓
Text Extraction
↓
AI Analysis
↓
ATS Score Generation
↓
Feedback & Suggestions
